INFO

Start #

对我来说，是惊喜、是开心、是回忆,感谢这一切的不期而遇和经历。一直都很庆幸运气不差，遇到公主~

与公主的见面 #

第一次见面

第一次见面，因为我个人原因导致堵车，未能第一时间去见面，让公主一个人待了一天，我心里一直都很愧疚，觉得很对不起，我也说过之后会对你越来越好。包括那次大吵架之后，你跟我说我对你不好，然后我开始跟你述说我的心结。跟你坦白，我为什么怕，你当时答应我说你不会的。 我其实真的会怕，但我也没多想什么，只是希望可以被你理解。之所以对你后来开始好是因为我只是希望和你在一起的时候，真的可以尽力去对你 每一次比上一次好一点。这也就是我说的 我会对你越来越好的原因之一。

  • 第一次去见公主的城市，快要出地铁站的时候排的，马上就要见到你了，真的超级激动。可能你觉得我很笨，不爱拍照、记录生活，是，我确实不怎么拍照，但是去见你前，我就在心里默默的告诉自己，好好的记录下每一次见面的时刻和故事。
  • 第一次和你晚上去路边散步(穿我喜欢的裙子)、去看电影、去一起吃火锅，当时你嫌弃我没有并排坐(其实我想说，坐在对面的寓意是心上人在眼前)，第一次去吃烤肉，(这次并排坐在你身边--寓意是心上人在身边)。 一起去zoom为了看一下大熊猫、去逛商场、第一次拍合照(你说要用我的手机拍照，想给我留作纪念)。谢谢你给我给我带来的快乐。

第二次见面

端午节的时候, 穿着你买的衬衫一起去逛街，去看电影，去文殊院，去你曾经待过的地方走走，你带我去吃你喜欢吃的美食。晚上去楼顶给你拍照(原谅我的拍照技术很Low)，还有带我去小吃街吃冰粉。

TIP

你给我准备了礼物(钱包&补水)，真的超级开心，我全都记在心里。

第三次见面

你身体不舒服，但是还是来见我了，说实话心里挺心疼的，没有好好照顾你(一杯热水也没能给你)，一起追剧、去买奶茶。送你去地铁，看着你远去，很珍惜在你身边的每一刻。没想到回去的时候，运气好像差一点，被隔离。你跟我说，挺自责的(我没有怪过你，因为去见自己喜欢的人，不然怎样 对我来说 都是值得的)。

第四次见面

这次见面对我来说挺不是滋味的，真的很不容易，一点也不顺利，可能对你来说，是给我一个分手仪式吧，我真的一直忍不住会掉眼泪，其实我真的一点都不喜欢在你面前哭，真的绷不住。第一次吃到你的早餐。那天打车被你说，我在路边蹲了好久(我没想到会被说), 秋天的第一杯奶茶，我当时想的是，送奶茶没意思，之前你说想去 蛋糕，所以我一直记着然后就自己去买了带给你(没想到被你说了一顿，我不知道你冰箱放不下， 没想到准备东西会被说)，说实话，被说心里挺不是滋味的，人生第一次(带点小心思准备的东西会被说一顿)。 不管怎样，，都很谢谢你。

talk #

你好像不知道我为什么喜欢你的原因，刚开始到第一次吵架的时候， 当时，遇到你真的挺开心的。 对我来说，或许是上天给我的惊喜吧，喜欢上你是因为你的性格与我很相似，当时的你 和曾经的我一样 ，当时也很喜欢你然后顺便想拉你一把 希望可以给到你光。没想过要谈多久~

后来喜欢你是 从那次坦白开始，我真的开始很想和你走下去，心里一直都有你，希望最后的那个人是你 。 我也说过 会每一次比前一次对你更好一点。 相比之前，后来 我可能对你不好，但是每次都在努力比上一次对你更好一点。
